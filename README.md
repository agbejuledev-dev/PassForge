# PassForge 🔐

PassForge is a cryptographically secure password generator built with pure vanilla JavaScript and zero dependencies.

It uses the browser Crypto API (`crypto.getRandomValues()`) instead of Math.random() to generate secure passwords and includes advanced features like entropy-based strength calculation, character filtering, password history tracking, keyboard shortcuts, and guaranteed policy-compliant password generation.

The interface follows a cyberpunk-inspired dark theme with animated grid background and glowing green accents for a technical developer-focused aesthetic.
